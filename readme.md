FundFlow – Personal Finance & Cashflow Tracker
├─ Catchy Line
│ └─ Take Control of Your Money Flow, Effortlessly
│
├─ Problem Statement
│ ├─ Daily expenses are scattered and hard to track consistently
│ ├─ Lack of clear visibility on where money is going
│ ├─ Poor awareness of spending habits leads to overspending
│ └─ No simple tool to monitor personal cashflow in real-time
│
├─ Proposed Solution
│ ├─ A mobile app to track income, expenses, and overall cashflow
│ ├─ Real-time overview of spending across categories
│ ├─ Simple and intuitive logging for daily transactions
│ └─ Visual insights to help users understand financial habits
│
├─ Target Users
│ ├─ Students and young professionals managing daily expenses
│ ├─ Individuals wanting better control of personal finances
│ └─ Users looking for a simple, clean budgeting tool
│
├─ Core Features (MVP)
│ ├─ Add / edit / delete income and expense transactions
│ ├─ Categorize transactions (food, transport, bills, etc.)
│ ├─ Monthly summary (income vs expenses)
│ ├─ Simple charts (spending breakdown)
│ └─ Transaction history list with filters
│
├─ Future Features
│ ├─ Budget planning per category
│ ├─ Recurring transactions (salary, subscriptions)
│ ├─ Multi-wallet / account support
│ ├─ Export reports (PDF / CSV)
│ └─ Cloud sync + authentication
│
├─ Screens
│ ├─ Dashboard (balance + quick insights)
│ ├─ Add Transaction
│ ├─ Transactions (history + filter)
│ ├─ Analytics (charts + breakdown)
│ └─ Profile / Settings
│
└─ Tech Stack
    ├─ React Native (Expo)
    ├─ Expo Router
    ├─ Node.js / Express (API)
    ├─ MySQL / PostgreSQL
    └─ REST API + basic caching / optimization
